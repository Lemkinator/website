Windows Subsystem for Android™️ (WSA)

Windows-Subsystem für Android™️ (WSA)

Install Android Apps on Windows

Android Apps auf Windows installieren

Info

Windows Subsystem for Android™️ (WSA) allows to install and use Android apps on Windows.

Windows Subsystem for Android™️ (WSA) ermöglicht es, Android-Apps auf Windows zu installieren und zu nutzen.

Unfortunately, Microsoft is discontinuing support for the WSA. Therefore, the Amazon Appstore on Windows and all applications and games dependent on WSA will no longer be supported starting March 5, 2025.

Microsoft stellt leider den Support für das WSA ein. Daher wird der Amazon Appstore unter Windows und alle von WSA abhängigen Anwendungen und Spiele ab dem 5. März 2025 nicht mehr unterstützt.


Thanks to the dedicated community, there is already a solution to continue using WSA, namely with the Google Play Store instead of the Amazon Appstore.

Dank der engagierten Community gibt es jedoch bereits eine Lösung, um WSA weiterhin zu nutzen, und zwar mit dem Google Play Store anstelle des Amazon Appstores.


Before installing the new version of WSA, the old official version from the Microsoft Store must be uninstalled.

Bevor die neue Version von WSA installiert wird, muss die alte offizielle Version vom Microsoft Store deinstalliert werden.




Install WSA

WSA installieren



Check system type in Windows settings:

Systemtyp in den Windows-Einstellungen überprüfen:



Steps

  1. Download the zip file (WSA_XXXX.XXXXX.X.X_systemtype_Release-Nightly-MindTheGapps-XX.X-RemovedAmazon.7z)
  2. Unzip (e.g.: to "C:\users\%yourUsername%\WSA")
  3. Optional: Zip file can be deleted after unzipping
  4. Open the unzipped folder and double click on "Run.bat"
  5. Confirm administrator rights
  6. Close the window when "All Done! Press any key to exit"
  7. In the dialog: "Optional Diagnostic Data" click on "Next". (Multiple of these dialogs are no problem, just click on "Next" every time)

Schritte

  1. Zip herunterladen (WSA_XXXX.XXXXX.X.X_systemtyp_Release-Nightly-MindTheGapps-XX.X-RemovedAmazon.7z)
  2. Zip entpacken (z.B.: nach "C:\Benutzer\%deinBenutzername%\WSA")
  3. Optional: Zip-Datei kann nach dem Entpacken gelöscht werden
  4. Entpackten Ordner öffnen und doppelklicken auf "Run.bat"
  5. Administratorrechte bestätigen
  6. Fenster kann geschlossen werden bei "All Done! Press any key to exit"
  7. Im Dialog: "Optionale Diagnosedaten" auf "Weiter" klicken. (Mehrere dieser Dialoge sind kein Problem, einfach immer auf "Weiter" klicken)



Google Play Store

A Google account is required to install Android apps.

Um Android Apps zu installieren, wird ein Google-Konto benötigt.

If the Google Play Store is not opened after installation of WSA, it can also be opened manually:

Falls der Google Play Store nicht nach der Installation von WSA geöffnet wird, kann dieser auch manuell geöffnet werden:

After opening the Google Play Store, a login with a Google account is requested.

Nach dem Öffnen des Google Play Stores wird die Anmeldung mit einem Google-Konto angefordert.

If the login was successful, Android apps can now be installed.

Wenn die Anmeldung erfolgreich war, können nun Android Apps installiert werden.

Access Windows Files in WSA

In WSA auf Windows-Dateien zugreifen

It is possible to access files of the Windows system within the WSA. To do this, access to a Windows folder must be enabled in the WSA settings.

Es ist möglich innerhalb des WSA auf Dateien des Windows-Systems zuzugreifen. Dazu muss in den WSA Einstellungen der Zugriff auf einen Windows-Ordner aktiviert werden.
